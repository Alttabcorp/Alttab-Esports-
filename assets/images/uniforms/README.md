# Imagens dos Uniformes

Este diretório contém as imagens dos uniformes do time para exibição no carrossel.

## Arquivos Necessários

Adicione as seguintes imagens neste diretório:

- `uniform-1.png` - Uniforme Principal
- `uniform-2.png` - Uniforme Alternativo
- `uniform-3.png` - Uniforme de Treino
- `jacket.png` - Jaqueta Oficial

## Especificações Recomendadas

- **Formato**: PNG ou JPG
- **Dimensões**: 800x1000px (proporção 4:5) ou similar
- **Tamanho**: Máximo 500KB por imagem para otimizar carregamento
- **Fundo**: Preferencialmente transparente (PNG) ou fundo neutro

## Dicas de Qualidade

- Use imagens de alta resolução mas otimizadas
- Mantenha o mesmo estilo/enquadramento em todas as fotos
- Se fotografar, use fundo neutro e iluminação adequada
- Considere usar mockups profissionais de uniformes

## Placeholder

Enquanto as imagens reais não estiverem disponíveis, o sistema usa um placeholder automático.
